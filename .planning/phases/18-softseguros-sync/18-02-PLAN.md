    - async get_poliza(poliza_id) → GET /api/poliza/{id}
    - async get_cliente(cliente_id) → GET /api/cliente/{id}
    - async close() → closes the underlying httpx.AsyncClient
    - All methods raise SoftSegurosAPIError (custom exception) on non-recoverable failures
  </behavior>
