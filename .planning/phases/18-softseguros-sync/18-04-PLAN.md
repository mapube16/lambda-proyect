  <name>Task 1: verify_pagopoliza_fresh function</name>
  <files>backend/softseguros/verify.py</files>
  <behavior>
    async def verify_pagopoliza_fresh(db, user_id, debtor_id) -> dict
    - Looks up debtor doc by _id in `debtors` collection, filtered by user_id
    - If not found or source != 'softseguros' or softseguros_pagopoliza_id missing → 404-like raise NotFoundError
    - Try: adapter = build adapter from credentials, call adapter.get_pagopoliza(softseguros_pagopoliza_id)
    - On 200 response:
      * If comisionada=true OR fecha_recibo_comision present:
          mark_debtor_paid_by_softseguros_id; return {should_call: false, reason: "already_paid", fresh_data: {...}}
      * Else if fecha_pago != local.fecha_pago OR valor_a_pagar != local.valor_a_pagar:
          upsert with new values (re-classify); return {should_call: true, reason: "outdated", fresh_data: {...}}
      * Else:
          update only last_verified=now; return {should_call: true, reason: "ok"}
    - On 404 from SOFTSEGUROS:
          mark_debtor_deleted_by_softseguros_id; return {should_call: false, reason: "not_found"}
    - On httpx.TimeoutException, 5xx after retries, or any unexpected error:
          return {should_call: true, reason: "ok", warning: "verification_unavailable"}  # fail-open
    - In all success paths, append a softseguros_sync_logs entry with mode='pre_call_check', total_requests=1
  </behavior>
