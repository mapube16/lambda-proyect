```python

```

Replace it with:
```python
    try:
        async with AsyncSession(
            impersonate="chrome131",
            allow_redirects=True,
            timeout=timeout,
        ) as client:
            hard_blocked = False
            for candidate in candidates:
                if hard_blocked:
                    break
                try:
                    resp = await client.get(candidate)
                except RequestsError as e:
                    last_error = f"{candidate}: {e}"
                    continue

                if resp.status_code >= 400:
                    last_error = f"{candidate}: HTTP {resp.status_code}"
                    if resp.status_code in _HARD_BLOCK_CODES:
                        hard_blocked = True
                    if resp.status_code in _NO_RETRY_CODES:
                        break
                    continue

                html = resp.text or ""
                if html.strip():
                    break
    except Exception as e:
        last_error = str(e)
```

Key changes:
- `httpx.AsyncClient(...)` → `AsyncSession(impersonate="chrome131", allow_redirects=True, timeout=timeout)`
- The inner `for headers in ua_profiles:` loop is removed entirely — curl_cffi generates authentic browser headers automatically from the impersonate target; manually passing headers degrades the fingerprint
- `except Exception as e:` on the inner try → `except RequestsError as e:` (curl_cffi-specific exception)
- `client.get(candidate, headers=headers)` → `client.get(candidate)` (no manual headers)
- The `if html.strip(): break` that was inside `for headers` is now directly after the candidate loop body

Do NOT modify ua_profiles — leave it in the function even though it is no longer used (harmless local variable).
