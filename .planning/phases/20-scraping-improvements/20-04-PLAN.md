Add the industria guard BETWEEN `n = int(max_r) if max_r else max_results` and `logger.info(...)`:

```python
        n = int(max_r) if max_r else max_results

        # Guard: if the LLM passed a generic term instead of the real campaign industry, override it.
        # Root cause: the director LLM sometimes infers industry from dolor_operativo context
        # rather than copying industria_objetivo literally.
        _GENERIC_INDUSTRY_TERMS = frozenset({
            "empresas", "empresa", "negocios", "negocio",
            "companies", "company", "businesses", "business",
            "",
        })
        campaign_industria = str(campaign.get("industria_objetivo") or "").strip()
        if campaign_industria and industria.strip().lower() in _GENERIC_INDUSTRY_TERMS:
            logger.warning(
                "[discover_companies] LLM passed generic industria=%r; "
                "overriding with campaign industria_objetivo=%r",
                industria,
                campaign_industria,
            )
            industria = campaign_industria

        logger.info(
            "[discover_companies] user=%s industria=%s ciudad=%s max=%d",
            user_id, industria, ciudad, n
        )
```

`campaign` is available via closure — it is the `campaign` parameter of `make_prospecting_registry` which is in scope for all inner functions including `_discover_companies`.
