    1. Tool dispatch helper:
       ```python
       async def dispatch_tool(name: str, params: dict, call_obj: dict) -> str:
           # Extract debtor_id: prefer params, fallback to variable_values in assistantOverrides
           debtor_id = (params.get("debtor_id") or
                       call_obj.get("assistantOverrides", {}).get("variableValues", {}).get("debtor_id"))
           db = get_db()
           try:
               if name == "consultar_deuda":
                   debtor = await db.debtors.find_one({"_id": ObjectId(debtor_id)}) if debtor_id else None
                   if not debtor:
                       return "Deudor no encontrado en el sistema."
                   return f"Deuda: ${debtor['monto']:,.0f} COP. Vencimiento: {debtor['vencimiento'].strftime('%d/%m/%Y')}."

               elif name == "registrar_promesa":
                   monto_prom = params.get("monto_prometido")
                   fecha_prom = params.get("fecha_prometida")
                   if debtor_id:
                       await db.debtors.update_one(
                           {"_id": ObjectId(debtor_id)},
                           {"$set": {"estado": "promesa_de_pago",
                                     "monto_prometido": monto_prom,
                                     "fecha_promesa": fecha_prom,
                                     "updated_at": datetime.now(timezone.utc)}}
                       )
                   return f"Promesa registrada: ${monto_prom} para {fecha_prom}. Gracias."

               elif name == "escalar_a_humano":
                   motivo = params.get("motivo", "")
                   if debtor_id:
                       await db.debtors.update_one(
                           {"_id": ObjectId(debtor_id)},
                           {"$set": {"estado": "escalado", "escalado": True,
                                     "updated_at": datetime.now(timezone.utc)}}
                       )
                   return "Escalado a agente humano. Un asesor le contactará pronto."

               else:
                   return "Herramienta no reconocida."
           except Exception as e:
               logger.error("[webhook] dispatch_tool error: %s", e)
               return "Error interno al procesar la herramienta."
       ```

    2. POST /api/vapi/tool-call endpoint:
       - Always wrapped in try/except at top level → always return JSONResponse({"results": results}, status_code=200)
       - Parse body.message.toolWithToolCallList
       - For each item: await dispatch_tool(name, params, call_obj) → append {"toolCallId": id, "result": str}
       - result MUST be string (not dict) — already guaranteed by dispatch_tool

    3. POST /api/vapi/call-ended endpoint:
       - Check message.type == "end-of-call-report" — if not, return JSONResponse({"ok": True})
       - Find debtor by vapi_call_id — if not found, return {"ok": True}
       - Map endedReason to new_estado:
         * "no-answer"|"busy"|"voicemail" → "sin_contacto"
         * "customer-ended-call"|"assistant-ended-call"|"hangup" → keep existing (tool calls already set it)
         * other → "fallido"
         * If current estado == "promesa_de_pago" → keep "promesa_de_pago" (overrides endedReason mapping)
         * If current estado == "escalado" → keep "escalado"
       - Check if new intentos >= debtor.max_intentos: override new_estado = "agotado"
       - Build call_record dict with call_id, fecha, duracion_segundos (from message.durationSeconds or 0),
         resultado=endedReason, transcript=artifact.transcript[:2000], recording_url=artifact.recordingUrl
       - Update debtor: $set estado/updated_at/ultimo_contacto_fecha, $inc intentos:1,
         $push historial_llamadas:call_record, $unset vapi_call_id:""
       - Import manager from main (lazy inside function to avoid circular import):
         ```python
         from main import manager  # lazy import inside function body
         await manager.send_to_user(str(debtor["user_id"]), {
             "type": "debtor_update",
             "debtor_id": str(debtor["_id"]),
             "estado": new_estado,
             "intentos": debtor.get("intentos", 0) + 1,
         })
         ```
       - Return JSONResponse({"ok": True})
