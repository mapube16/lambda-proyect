       - Fetch empresa_nombre from get_client_profile(user_id) — use profile.get("empresa_nombre", "la empresa")
       - Call generate_cobranza_proposal(descripcion, empresa_nombre) — asyncio.create_task NOT needed here (it's fast)

    2. POST /api/cobranza/onboarding/approve
       - Body: {"estrategia": dict} — the proposal (possibly user-edited)
       - Upsert into db.cobranza_config with user_id as key:
         ```python
         await db.cobranza_config.update_one(

             {"$set": {"estrategia": estrategia, "updated_at": datetime.now(timezone.utc)}},
             upsert=True
         )
         ```
       - Return 200 {"campaign_id": user_id, "ok": True}

    3. POST /api/cobranza/debtors/{debtor_id}/llamar-ahora
       - Auth required
       - Check is_contact_allowed_now() — if False: return 400 {"error": "Fuera de horario permitido (Ley 2300)"}
       - Check has_been_contacted_today(debtor) — if True: return 400 {"error": "Ya fue contactado hoy (Ley 2300)"}
       - Fetch debtor — 404 if not found or wrong user_id
       - Fetch campaign config from db.cobranza_config
       - Update debtor estado to "llamando", set vapi_call_id to None temporarily
       - asyncio.create_task(initiate_call_and_update(debtor, config)) — fire and forget
       - Return 202 {"ok": True, "message": "Llamada iniciada"}

       Helper coroutine initiate_call_and_update(debtor, config):
       - Call initiate_call(debtor, config) from vapi_client
       - On success: update debtor vapi_call_id = call_id
       - On RuntimeError: update debtor estado back to "pendiente", log error
