   ```
   cd backend && source venv/Scripts/activate && python --version
   ```

4. Install hive tools first (workspace dependency workaround), then core:
   ```
   pip install -e ../vendor/hive/tools
   pip install -e ../vendor/hive/core
   ```
   If `tools` directory does not exist in the repo, skip the tools step and install core only.
   If `uv` is available, prefer: `uv pip install -e ../vendor/hive/core`

   ```
   python -c "from framework.runner.runner import AgentRunner; import framework; print(framework.__file__)"
   ```

   ```
   pip check
   ```
   Conflicts involving `openai==1.12.0` vs `litellm` are expected — note but do not resolve in this plan.
