- `async def _embed_text(text: str) -> list[float]` — usa AsyncOpenAI, model=EMBED_MODEL. Trunca texto a 8000 chars si excede. Retorna `response.data[0].embedding`.

**Chunking (per D-07, D-09):**
- `def _chunk_text(text: str) -> list[str]` — `RecursiveCharacterTextSplitter(chunk_size=1000, chunk_overlap=100, length_function=len, is_separator_regex=False).split_text(text)`. Filtra chunks vacíos.

**`async def ingest_document(user_id: str, content: str, title: str, doc_type: str) -> dict`:**
1. `assert user_id, "user_id required for Pinecone namespace isolation"` — CRÍTICO, primera línea
2. Si no PINECONE_API_KEY: return `{"doc_id": None, "chunk_count": 0, "error": "PINECONE_API_KEY not set"}`
3. chunks = `_chunk_text(content)`. Si len == 0: return `{"doc_id": None, "chunk_count": 0, "error": "no chunks"}`
4. Generar `doc_id = str(uuid.uuid4())`
5. Embeds en batch: `vectors = []`; para cada i, chunk en enumerate(chunks): `emb = await _embed_text(chunk)`; `vectors.append({"id": f"{doc_id}_chunk_{i}", "values": emb, "metadata": {"user_id": user_id, "doc_id": doc_id, "title": title, "doc_type": doc_type, "chunk_index": i, "text": chunk[:500]}})`
6. `index = await _get_pinecone_index()`; `await index.upsert(vectors=vectors, namespace=user_id)` — SIEMPRE namespace=user_id
7. `from cobranza.tenant_config import save_rag_document_metadata`; `await save_rag_document_metadata(user_id, title, doc_type, len(chunks))`
8. return `{"doc_id": doc_id, "chunk_count": len(chunks)}`

**`async def search_knowledge(user_id: str, query: str, top_k: int = 5) -> list[dict]`:**
1. `assert user_id, "user_id required for Pinecone namespace isolation"` — CRÍTICO, primera línea
2. Si no PINECONE_API_KEY o no query: return `[]`
3. `query_vector = await _embed_text(query)`
4. `index = await _get_pinecone_index()`; si index None: return `[]`
5. `result = await index.query(vector=query_vector, top_k=top_k, namespace=user_id, include_metadata=True)` — SIEMPRE namespace=user_id, NUNCA None
6. return `[{"text": m["metadata"].get("text", ""), "score": m["score"], "title": m["metadata"].get("title", ""), "doc_type": m["metadata"].get("doc_type", "")} for m in result.get("matches", [])]`

logger = `logging.getLogger("cobranza.rag_service")`
